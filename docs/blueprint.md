# **App Name**: Controle de doses

## Core Features:

- Patient Registration: Register new patients with complete details: full name, age, initial weight, height, desired weight, first dose date, address with automatic ZIP code lookup, phone number for WhatsApp contact, and health contraindications form.
- Automatic BMI Calculation: Automatically calculate the BMI based on height and weight during patient registration and dose management.
- Dose Schedule Generation: Automatically generate a 12-dose schedule for the treatment, with each dose 7 days after the previous one.
- Dose Management: Manage doses with options to view current weight, automatic BMI calculation, administered dose (2.5 / 3.75 / 5 / 6.25 / 7.05mg), payment method selection (cash, Pix, debit, credit, payment link), and installment options for credit payments (2-12 installments).
- Dose Rescheduling: Allow rescheduling of doses, recalculating subsequent doses based on the new date.
- AI-Powered Personalization Tool: AI-driven assistant tool to review and incorporate the latest personalized guidelines related to dosage or timing, potentially optimizing the user's experience based on similar situations, using real-time data ingestion from health advisory websites.
- Dose Reminder System: Implement a color-coded dose reminder system: green (more than 5 days until the next dose), yellow (3 days until the dose), and red (1 day or dose expired).

## Style Guidelines:

- Primary color: Light blue (#A7D9ED), conveying calmness and health.
- Background color: Very light blue (#F0F8FF), offering a clean and unobtrusive backdrop.
- Accent color: Soft green (#B0E57C), signaling progress and health-related actions.
- Body and headline font: 'PT Sans', a modern humanist sans-serif suitable for both headlines and body text.
- Use clean, modern icons to represent actions and data points. Icons should be consistent and easily understandable.
- Design a clear and intuitive layout that facilitates easy navigation and data input. Prioritize key information and actions for quick access.
- Incorporate subtle animations to enhance user interaction and provide feedback. Focus on smooth transitions and loading indicators to create a seamless experience.
